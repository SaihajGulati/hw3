For Parts 1 and 2, compile my llrec and tester with 
```
make
```
Clean object and other temporary files with 
```
make clean
```
Run my tester with 
```
./llrec-test llrec-test1.in
```

For part 3, compile my stack test with 
```
g++ -g -Wall --std=c++11 stack_test.cpp -o test
```
And run with
```
./test
```
